Hexo+Github部署静态博客网站

声明：本文节选自如何使用 github pages + Hexo + Next 搭建个人博客，更复杂的功能可以参考Hexo的Next主题优化教程

创建Github仓库：

  1. 创建以<username>.github.io命名的public github仓库；

安装配置Hexo项目：

  1. 在本地安装Hexo框架：npm install -g hexo-cli；
  2. 初始化一个Hexo项目：hexo init myBlogs；
  3. 进入项目目录安装依赖：cd myBlogs npm install；
  4. 启动Hexo服务：hexo server，浏览器访问http://localhost:4000/验证安装是否成功；

更换Next主题：

  1. 进入项目目录克隆Next主题文件到themes文件夹：

    git clone https://github.com/theme-next/hexo-theme-next themes/next

  2. 将_config.yml文件中的主题参数，设置theme: next

  3. 启动Hexo服务，验证Next主题是否适用：hexo server

  4. 通过在文章想要折叠的部分加<!-- more -->可以控制折叠，感谢This Blog

将页面部署到Github：

  1. 修改_config.yml：

  2. 安装deploy-git：npm install hexo-deployer-git --save

如何加载图片：

  1. 默认根路径是source文件夹，可以在source文件夹下新建figures文件夹存放图片，在md文件中使用:

    <img src="/figures/<fig_name>.jpg">引用即可。
